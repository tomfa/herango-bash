Bash

Herango-Bash is just a bash-script. It helps you quickly setup Django and push it to Heroku. Real fast. SuperASAP.

As long as you're connected to the internet, you can always run this script with the following command via the terminal: curl -O https://raw.githubusercontent.com/tomfa/herango-bash/master/install.sh ; sh install.sh
